Geo Tags Austria

== Description ==

Mit Geo Tags Austria koennen die Angaben geo.placename, geo.position, geo.region and ICBM einzeln fuer Austria eingestellt werden. Support: [http://www.tagesgeld.ag/](http://www.tagesgeld.ag/)

== Installation ==

1. Extract `geo-tags-austria.1.0.zip`.
2. Upload the `geo-tags-austria`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
